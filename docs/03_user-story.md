# User Stories

### US-1. Введення параметрів
**Як** `Guest` (неавторизований користувач)  
**Я хочу** ввести свої параметри (вік, вага, зріст, стать, рівень активності)  
**Щоб** `програма` могла розрахувати мої базові потреби без прямого доступу до логіки розрахунку.  

### US-2. Вибір мети
**Як** `Guest`  
**Я хочу** вказати свою ціль (схуднення, підтримка, набір маси)  
**Щоб** `програма` скоригувала калораж та БЖВ відповідно до обраної мети.  

### US-3. Детальний розрахунок (БЖВ)
**Як** `Guest`  
**Я хочу** бачити не лише калорії, а й рекомендовані **білки, жири, вуглеводи**  
**Щоб** `програма` обрахувала грамові значення відповідно до правил обраної дієти.  

### US-4. Генерація раціону
**Як** `Guest`  
**Я хочу** натиснути кнопку "Згенерувати раціон"  
**Щоб** `програма` створила `план харчування` з 3-х прийомів їжі, які відповідають моїм нормам.  

### US-5. Перегляд складу страв
**Як** `Guest`  
**Я хочу** бачити БЖВ біля кожної страви 
**Щоб** розуміти, звідки надходять білки, жири та вуглеводи.
