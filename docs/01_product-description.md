# NutriPlan

## Опис

**NutriPlan** — це настільний застосунок, розроблений на платформі **WPF (.NET Framework 4.7.2)**, призначений для автоматичного розрахунку добової потреби в калоріях та макронутрієнтах на основі персональних даних користувача. Після виконання розрахунків система формує практичний одноденний план харчування, використовуючи локальну базу продуктів у форматі JSON.

---

## Основні можливості

### Автоматичний розрахунок калорій та БЖВ

Застосунок визначає добову норму калорій і розподіл білків, жирів та вуглеводів на основі таких параметрів:

- Вага
- Зріст
- Вік
- Стать
- Рівень фізичної активності
- Ціль користувача:
  - схуднення;
  - підтримка ваги;
  - набір маси.

Розрахунки реалізовані в сервісах:

- `CalorieCalculatorService`
- `MacroCalculatorService`

### Генерація денного раціону

На основі розрахованих показників система автоматично формує меню на один день, яке включає:

- 🍳 Сніданок
- 🍲 Обід
- 🍽️ Вечерю
- 🍎 Перекуси

Для цього використовується локальна база продуктів, що завантажується через `JsonProductRepository`, а розподіл калорій між прийомами їжі виконується сервісом `MealDistributionService`.
