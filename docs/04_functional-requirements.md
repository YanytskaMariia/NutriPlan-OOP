# Функціональні вимоги (Functional Requirements)

У цьому розділі описані основні функціональні можливості системи для розрахунку КБЖВ та генерації раціону харчування.

## 📝 FR1: Введення персональних даних гостем
**Опис:** Гість має можливість ввести або обрати свої фізіологічні параметри для подальших розрахунків.  
**Вхідні дані:** Стать, вага (кг), зріст (см), вік (років), рівень фізичної активності.

**✅ Критерії прийняття:**
- [ ] Всі поля (стать, вага, зріст, вік, активність) доступні для заповнення/вибору.
- [ ] Система валідує введені дані (наприклад, вага не може бути від’ємною).
- [ ] Після підтвердження введення, дані тимчасово зберігаються у системі для розрахунку.

---

## 🎯 FR2: Вибір мети користувача
**Опис:** Гість обирає одну з цілей для корекції або підтримки ваги.  
**Варіанти:** Схуднення, Підтримка ваги, Набір маси.

**✅ Критерії прийняття:**
- [ ] Користувач може обрати лише одну мету одночасно.
- [ ] Обрана мета змінює коефіцієнти при розрахунку добової норми КБЖВ (дефіцит, норма або профіцит).

---

## 🧮 FR3: Розрахунок добової норми КБЖВ
**Опис:** Система автоматично обчислює необхідну добову норму калорій, білків, жирів та вуглеводів на основі введених даних (FR1) та мети (FR2).

**✅ Критерії прийняття:**
- [ ] Розрахунок ініціюється автоматично після завершення введення даних.
- [ ] Використовується затверджена формула розрахунку (наприклад, Міффліна-Сан Жеора).
- [ ] Результат розрахунку готовий до відображення без затримок.

---

## 📊 FR4: Перегляд згенерованої добової норми
**Опис:** Гість бачить на екрані деталізований результат розрахунку.

**✅ Критерії прийняття:**
- [ ] Відображаються конкретні числові значення: Калорії (ккал), Білки (г), Жири (г), Вуглеводи (г).
- [ ] Інформація подана у зрозумілому та зручному для сприйняття форматі.

---

## 🍽️ FR5: Генерація денного раціону
**Опис:** За запитом користувача ("Згенерувати раціон") система формує план харчування на один день.

**✅ Критерії прийняття:**
- [ ] Раціон складається з 3-х прийомів їжі (Сніданок, Обід, Вечеря).
- [ ] Сумарна калорійність та БЖВ згенерованого раціону відповідає розрахованій добовій нормі з похибкою не більше ±10%.
- [ ] Згенерований список страв відображається на екрані одразу після завершення процесу.
